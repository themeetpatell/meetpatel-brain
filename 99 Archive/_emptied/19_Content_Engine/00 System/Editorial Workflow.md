# Editorial Workflow

## Daily
- capture one real observation
- choose one angle worth publishing
- write one strong final post
- generate one X version
- create one quick repurpose pack

## Weekly
- review what worked
- identify repeated themes
- elevate strong ideas into bigger assets
- build next week’s queue

## Monthly
- see which themes are strengthening brand perception
- cut weak recurring patterns
- sharpen positioning through published work
