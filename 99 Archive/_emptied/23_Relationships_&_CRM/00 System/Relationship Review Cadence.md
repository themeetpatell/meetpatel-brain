# Relationship Review Cadence

## Weekly
- review follow-up queue
- log new meaningful interactions
- identify one relationship to deepen

## Monthly
- update warm network tracker
- revive 3 dormant relationships
- add strategic notes to key people

## Quarterly
- map top relationship gaps
- identify new people worth knowing
- review which relationships are compounding
