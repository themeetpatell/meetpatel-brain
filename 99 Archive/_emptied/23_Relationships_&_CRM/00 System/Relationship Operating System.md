# Relationship Operating System

## Purpose
Turn your network into a living strategic asset instead of a scattered contact list.

## Principles
- nurture before you need
- depth beats random volume
- relevance matters more than frequency
- one meaningful touchpoint compounds more than ten empty messages
- document context so trust does not depend on memory

## Core actions
1. map people by category
2. identify strategic value and relationship strength
3. maintain a follow-up rhythm
4. log meaningful context
5. convert warm relationships into trusted long-term allies
