# AI Operating System

## Purpose
Use multiple AI systems as a coordinated leverage layer across thinking, execution, research, and writing.

## Core layers
- model selection
- prompt design
- workflow orchestration
- output evaluation
- reuse and memory

## Rule
Do not treat models as magic. Treat them as operators with strengths, limits, and costs.
