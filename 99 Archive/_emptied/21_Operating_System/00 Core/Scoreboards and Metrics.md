# Scoreboards and Metrics

## Personal
- days with deep work
- content published
- sleep quality
- fitness consistency
- weekly reflection completed

## Venture
- priority completion
- key GTM metrics
- product milestones
- cash/revenue markers
- team execution markers

## Rule
If a metric does not influence behavior, remove or redesign it.
