# 70 Meta — dashboards, templates, system files

> Operating surface. Dashboards humans look at, templates new notes inherit, docs about the vault itself.

## Subfolders
- `Dashboards/` — Home, Today, Founder Dashboard, Venture Dashboard, Energy Dashboard
- `Templates/` — note templates (Daily, Weekly, Venture, Decision, etc.)
- `Scripts/` — vault automation scripts (Templater, Dataview JS)
- `Docs/` — docs about the vault itself (superpowers guide, MOC philosophy)

## Note
- MOCs are NOT here — they live in `_AI/MOCs/` (single home for navigation)
- Dev code (Node.js scripts) lives in `.scripts/` at vault root

## How agents use this folder
- Read `Dashboards/Today.md` to see current focus
- Read `Templates/` to know the canonical shape for new notes
- Skip everything else
