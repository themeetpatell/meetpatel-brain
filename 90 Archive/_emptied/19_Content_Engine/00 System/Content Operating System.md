# Content Operating System

## Purpose
Turn scattered founder thinking into a repeatable publishing engine.

## Flow
1. Capture raw insight
2. Convert into angle
3. Draft strongest version
4. Adapt across platforms
5. Publish
6. Review performance
7. Feed learnings back into the system

## Content priorities
- clarity
- signal
- distinctiveness
- consistency
- repurposability

## Core channels
- LinkedIn
- X
- Newsletter
- WhatsApp / broadcast
- optional short-form video
