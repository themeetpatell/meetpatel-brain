# Execution Cadence

## Daily
- choose top priorities
- protect deep work
- capture decisions
- close loops

## Weekly
- review wins and waste
- reset next moves
- update key trackers
- strengthen one system

## Quarterly
- define priorities
- cut noise
- reallocate time and attention
- reinforce the highest-compounding bets
