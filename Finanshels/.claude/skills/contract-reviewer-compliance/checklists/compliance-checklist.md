# Contract Compliance Checklist — Finanshels UAE

> Use this checklist for every contract review. Work through each section in order.
> Mark each item: ✅ PASS | ⚠️ FLAG | ❌ FIX REQUIRED | N/A (not applicable to this contract type)

---

## A. Governing Law & Jurisdiction

| # | Check | Status | Notes |
|---|-------|--------|-------|
| A1 | Governing law clause is present | | |
| A2 | Governing law is UAE (mainland courts, DIFC Courts, or named free zone forum) | | |
| A3 | If foreign governing law: risk is flagged as HIGH and legal advice sought | | |
| A4 | Dispute resolution mechanism is specified (arbitration / DIFC Courts / onshore courts) | | |
| A5 | Arbitration seat and rules are named if arbitration is chosen (e.g. DIAC, ICC) | | |

## B. Scope of Services

| # | Check | Status | Notes |
|---|-------|--------|-------|
| B1 | Scope is specific and exhaustive — no open-ended catch-all language | | |
| B2 | Services outside scope are explicitly excluded (e.g. legal advice, investment advice) | | |
| B3 | For CT/VAT work: scope states Finanshels acts as agent/authorized representative, not guarantor | | |
| B4 | Deliverables and timelines are defined or referenced in a schedule | | |
| B5 | Change-of-scope process is defined (written variation orders) | | |

## C. Liability & Indemnity

| # | Check | Status | Notes |
|---|-------|--------|-------|
| C1 | Liability cap is present | | |
| C2 | Cap is set at fees paid in prior 12 months OR a fixed AED amount — not uncapped | | |
| C3 | Consequential, indirect, and special loss are excluded | | |
| C4 | Mutual indemnity obligations do not expose Finanshels to client-caused regulatory penalties | | |
| C5 | Professional indemnity insurance requirement is stated and achievable | | |
| C6 | Force majeure clause is present and covers regulatory changes | | |
| C7 | No provision making Finanshels a guarantor of tax outcomes or refunds | | |

## E. AML/CFT & DNFBP Obligations

| # | Check | Status | Notes |
|---|-------|--------|-------|
| E1 | Confidentiality clause contains a carve-out for legal/regulatory obligations | | |
| E2 | No clause restricts Finanshels from filing SAR/STR reports on the goAML portal | | |

| # | Check | Status | Notes |
