# Ad Formats Reference

> Character limits and format specs per platform. Always count characters
> including spaces. Copy must fit with NO truncation. Specs change — confirm
> against the current platform help centre before a major launch.

---

## Google Search — Responsive Search Ad (RSA)

| Element | Limit | Count | Notes |
|---------|-------|-------|-------|
| Headline | 30 chars | 8-15 (min 3 shown) | Front-load keywords; ≥3 benefit-led; 1 CTA headline |
| Description | 90 chars | 2-4 | Expand the offer + urgency |
| Display path | 15 chars each | 2 fields | After domain, e.g. /vat-uae /register |
| Final URL | — | 1 | Landing page with matching message |

Best practice:
- Pin the brand name to one headline position only if needed; otherwise leave unpinned.
- Include the urgency hook (real FTA deadline) in at least one description.
- Mirror the searcher's keyword in headline 1.

## Meta — Instagram / Facebook (Feed & Lead Ads)

| Element | Limit | Notes |
|---------|-------|-------|
| Primary text | 125 chars visible before "more" (write ≤600 total) | Lead with Pain in the first line |
| Headline | 40 chars | The bold line under the image |
| Description / link description | 30 chars | Often shown on link ads |
| CTA button | Preset | Book Now, Learn More, Get Offer, Sign Up, Contact Us |

Best practice:
- The first ~125 characters are everything — put the hook there.
- One idea per ad; conversational, founder-first.
- Emoji: 0-2 max, only if on-brand; never in place of words.
- Carousel: each card needs its own headline (40) + description (30).

## LinkedIn — Single Image / Document Ads (Sponsored Content)

| Element | Limit | Notes |
|---------|-------|-------|
| Intro text | 150 chars before "see more" (≤600 total) | Credible, peer-to-peer tone |
| Headline | 70 chars | Clear value statement |
| Description | 70 chars | Shown in some placements |
| CTA button | Preset | Learn More, Request Demo, Sign Up, Register, Download |

Best practice:
- Audience is finance leads and founders — no hype, no exclamation stacks.
- Lead with a business outcome, not a feature.
- Document ads suit lead-magnet promotion (checklists, guides).

## Cross-platform brand rules

- Formula on every ad: **Pain → Solution → Proof → CTA**.
- Lead **WHY → WHAT** — never open with "We are an accounting firm".
- Proof point per ad (use exact figures): 7,000+ businesses served · 150+ qualified accountants · AI-native delivery · Trusted by Abwaab, Silkhause, Dubai Future District Fund, Cotu Ventures.
- Voice: founder-first ("you/your"), plain English, no filler ("synergies", "leverage", "holistic").
- No guarantees ("never pay a penalty", "100% approval").
- Urgency = real FTA deadlines only (e.g. CT return due 9 months after FY-end; VAT return due 28th of the following month).
- CTA buttons: brand orange **#F16610**. Headers/dark text: Deep Blue **#0A2B3B**. Font: Inter.

## Variation matrix (for A/B testing)
