**Classification guide:**
- **CRITICAL** — Workpaper cannot be used to file a return or deliver to an auditor until resolved. FTA penalty or compliance risk.
- **MATERIAL** — Significant quality issue that must be resolved before sign-off.
- **MINOR** — Best-practice improvement; does not block sign-off but should be addressed.
