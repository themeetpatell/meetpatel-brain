# 00 Inbox — raw capture, triage zone

> Everything new lands here. Nothing lives here.

## Rules
- Capture fast, no formatting required
- File or kill within 7 days (weekly review)
- Anything older than 14 days → either promoted to a real zone or moved to `99 Archive/`

## Sources that drop in here
- iOS Shortcut → `capture.sh`
- Voice notes → transcripts
- Notion imports → `notion/` subfolder

## Triage targets
| If it's… | It goes to… |
|---|---|
| A daily reflection | `10 Daily/Daily/<date>.md` |
| A venture idea/note | `30 Ventures/<name>/Notes/` |
| A book/framework/play | `50 Atlas/<zone>/` |
| A decision | `20 Compass/Decisions/` or `30 Ventures/<name>/Decisions/` |
| Personal life | `40 Areas/<zone>/` |
| Outdated/dead | `99 Archive/` |

## How agents use this folder
- Don't read raw inbox files for context — they're unrefined
- DO scan filenames weekly to surface what needs triage
