## Section A — AML/CFT Customer Due Diligence (CDD)

> Mandatory. Finanshels is a DNFBP. CDD must be completed before work begins.

### A1. Engagement Letter
- [ ] Signed engagement letter / service agreement on file
- [ ] Scope, fees, and payment terms confirmed in writing

### A2. Entity Identification
- [ ] Trade licence collected (valid, unexpired)
- [ ] Certificate of Incorporation collected
- [ ] MoA / AoA collected
- [ ] Registered address confirmed

### A3. Beneficial Ownership
- [ ] Passport copies collected for all shareholders ≥ 25%
- [ ] Emirates ID copies collected (if UAE resident)
- [ ] UBO declaration form completed and signed by client
- [ ] UBO register extract obtained (from free zone authority or registrar)
- [ ] Nominee director confirmation (if applicable)

### A4. Authorized Signatories
- [ ] Passport + EID for all authorized signatories
- [ ] Board resolution / POA authorizing signatory (if not a shareholder)

### A5. Sanctions Screening
- [ ] Client entity name screened against UAE sanctions lists
- [ ] All UBOs and shareholders screened
- [ ] Screening tool / portal used: _______________
- [ ] Date of screening: _______________
- [ ] Result: Clear / Match found (escalate immediately if match)

### A6. Risk Rating
- [ ] Risk rating assigned: **Low / Medium / High**
- [ ] Rationale documented (industry, geography, ownership structure, PEP status)
- [ ] Enhanced Due Diligence (EDD) required? Yes / No
  - If Yes: EDD completed and signed off by: _______________ Date: _______________

### A7. CDD Sign-Off
- [ ] CDD file complete and reviewed
- [ ] Sign-off by engagement manager: _______________ Date: _______________

## Section B — Entity-Specific Documents

### B1. All Entities (Required)
- [ ] Trade licence (valid)
- [ ] Certificate of Incorporation
- [ ] MoA / AoA
- [ ] Passport + EID for all shareholders ≥ 25% and signatories
- [ ] UBO declaration / UBO register extract
- [ ] Proof of registered address (≤ 3 months old)

### B2. Free Zone Entities (Additional)
- [ ] Free zone licence + registration certificate
- [ ] Share register issued by free zone authority
- [ ] Licence category noted (trading / services / holding / manufacturing)
- [ ] Note: QFZP eligibility under Corporate Tax to be assessed separately

### B3. VAT (If VAT services engaged)
- [ ] VAT TRN certificate from FTA
- [ ] Last 3 VAT return PDFs (if transferring agent)
- [ ] VAT return filing frequency: Quarterly / Monthly
- [ ] EmaraTax agent access request sent: Date _______________
- [ ] EmaraTax agent access confirmed by client: Date _______________

### B4. Corporate Tax (If CT services engaged)
- [ ] CT registration certificate (if already registered on EmaraTax)
- [ ] If not yet registered: CT registration deadline noted (see FTA schedule by licence month)
- [ ] Last audited financial statements (or draft accounts)
- [ ] Existing chart of accounts / trial balance
- [ ] CT return due date calculated: _______________ (9 months after financial year-end)
- [ ] EmaraTax CT agent access confirmed: Date _______________

### B5. Bookkeeping (If bookkeeping services engaged)
- [ ] Prior accounting software identified: _______________
- [ ] Access granted / new file set up
- [ ] Opening trial balance (signed off date): _______________
- [ ] First bookkeeping period defined: From _______________ To _______________
- [ ] Chart of accounts reviewed and agreed
- [ ] Bank feeds connected (bank name + account): _______________

### B6. Payroll / WPS (If payroll services engaged)
- [ ] WPS-registered bank account details received
- [ ] Current employee list (name, salary, start date, nationality)
- [ ] Labour contracts reviewed (sample)
- [ ] MOHRE registration confirmed
- [ ] GPSSA obligation assessed (UAE/GCC national employees?)
- [ ] First payroll run date: _______________
