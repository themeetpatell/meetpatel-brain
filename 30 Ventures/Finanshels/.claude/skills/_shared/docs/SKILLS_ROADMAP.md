## 📋 Skill Template Structure

Each skill folder will contain:

```
SKILL_NAME/
├── SKILL.md (main definition)
├── examples.md (prompt examples)
├── workflows/ (step-by-step processes)
│   ├── workflow-1.md
│   ├── workflow-2.md
├── tools/ (Python utilities)
│   ├── tool-1.py
│   ├── tool-2.py
└── templates/ (reusable templates)
    ├── template-1.md
    └── template-2.md
```
