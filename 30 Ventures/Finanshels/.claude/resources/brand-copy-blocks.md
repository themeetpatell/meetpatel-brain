## ✅ Quality Checklist for Any Copy

Before publishing ANY Finanshels communication, verify:

- [ ] **WHY comes first** — Opens with purpose, not service description
- [ ] **Proof point included** — At least one (7,000+, 150+, etc.)
- [ ] **Founder-first language** — "You/your", not "clients/businesses"
- [ ] **Action-oriented** — Verb-led, tells what to do next
- [ ] **Specific, not generic** — Numbers, not flowery language
- [ ] **Plain English** — No jargon without translation
- [ ] **Regional authority** — UAE/GCC specific
- [ ] **Clear CTA** — Never end without directing reader
- [ ] **Brand tone matches channel** — Website ≠ TikTok ≠ Investor
- [ ] **Never "just accounting firm"** — Includes reliability + scale + tech

## 📋 Do's and Don'ts

### Do ✅
- Lead with WHY (financial clarity for founders)
- Use exact numbers (7,000+, 150+, $50B+)
- Position as infrastructure, not service
- Speak directly to founder pain
- Include one clear CTA
- Use regional language (UAE, GCC)
- Anchor in proof points

### Don't ❌
- Describe as "accounting firm" without qualifier
- Use jargon without explanation
- Make generic promises ("best service", "top quality")
- Use passive voice ("Services are provided")
- Bury the value proposition
- Be vague ("We help with finances")
- Over-promise ("We guarantee success")
- Use corporate filler ("synergies", "holistic")
