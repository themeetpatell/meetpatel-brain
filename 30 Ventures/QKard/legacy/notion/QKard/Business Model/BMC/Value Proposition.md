# Value Proposition

Created: May 18, 2025 2:42 PM

### USP :

**Product**

1. Lends credit for gaming purchase.**
2. More credits on every purchase.
3. Also allows gaming transactions under INR50.
4. Allows lending of game points.**
5. Buy coupons and shopping vouchers using game points collected at any platform.

**Security**

1. Fragment card with different amount. Each fragment will have different virtual address, so no card will be exposed.
2. Will have freedom to dispose the fragment after single use.**
3. Anti money laundering measures.
4. GDPR compliant, does not save any sensitive details.
