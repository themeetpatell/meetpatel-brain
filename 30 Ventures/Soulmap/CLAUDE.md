# Soulmap — Agent Context

> Read this before doing any work on Soulmap. Note: Soulmap's existence as a standalone venture is itself an open strategic question — read Open Questions before deep work.

## One-paragraph snapshot

Soulmap is a **relationship-intelligence concept** layering soul profiling, compatibility logic, identity-led matching on top of (or alongside) the Biggdate thesis. The bet: there's an underlying relationship-intelligence engine that powers Biggdate's matching and could also live as a standalone product / service / API / framework. Currently a thesis-stage venture with 4 legacy notes — Strategic Questions, Product Thesis, Positioning, Master Note.

## Why it matters (to Meet)

- Adjacent expansion path from Biggdate — if Biggdate wins, Soulmap is the IP layer that compounds
- Standalone option if Biggdate doesn't scale — relationship intelligence has broader applications (HR, team formation, coaching)
- Meet has deep personal interest in behavioral science + relationship dynamics — venture aligns with that
