# Framework Atlas — Map of Content

> 25 strategic frameworks distilled and made applicable to Meet's ventures. Each note: when to use → what it is → how to apply → anti-patterns → related. No textbook theory. Operator-ready only.

## How to use this atlas

- Stuck on a strategy / product / sales / ops / decision problem? Find the closest framework category.
- Each framework tells you (a) when it's the right tool, (b) the framework itself in 3–5 steps, (c) one Meet-applied example.
- If a framework doesn't have an applied example yet — pick a real venture problem and run it.
- Cross-link aggressively: many frameworks compose.

## Cross-framework playbooks (when frameworks combine)

- **New venture launch:** Working Backwards → JTBD → Wedge → MEDDIC qualification of first 10 prospects → AARRR funnel sketch
- **Stuck venture decision:** Premortem → 7 Powers audit → Type 1/Type 2 → Decision Frame
- **Growth stall:** Sean Ellis test → Mom Test interviews → Growth Loops audit → ICE-score new bets
- **Sales process build:** Bullseye → MEDDIC → SPIN → AARRR
- **Quarterly planning:** OKRs + 4DX scoreboard + EOS Rocks

## How to add a new framework

1. Create note in the right subfolder
2. Use this structure:
   - One-paragraph "what it is"
   - "When to use it" (the trigger)
   - The framework in 3–5 steps
   - "How to apply to [venture]" (at least one concrete Meet example)
   - Anti-patterns
   - Related frameworks
   - Source
3. Add wikilink to this MOC
4. Tag with `framework` + category

## Maintenance

- Quarterly: review which frameworks Meet has actually pulled from. The unused ones are noise.
- After every major decision / strategy work, retro: which framework would have helped? Add it if missing.
